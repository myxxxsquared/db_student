Redis的安装

Linux环境

1.下载并安装稳定版本

  1. $ wget http://download.redis.io/releases/redis-stable.tar.gz
  2. $ tar xzf redis-stable.tar.gz
  3. $ cd redis-stable
  4. $ make

2.启动Redis服务端,保持该终端不关闭

  1. $ cd src
  2. $ ./redis-server

3.开启一个新的终端，cd至相应src目录下，启动Redis客户端

  1. $ cd redis-stable
  2. $ cd src
  3. $ ./redis-cli

Windows环境

1.前往 Redis的GitHub页面 下载Redis-Windows最新版本3.0.504的Zip压缩包。
2.在C盘创建redis文件夹，将zip压缩包解压至该文件夹。
3.开启一个cmd窗口,切换至C:\redis，启动Redis服务端，保持该cmd窗口不关闭

  1. cd C:\redis
  2. redis-server.exe redis.windows.conf

4.再开启一个cmd窗口，切换至C:\redis，启动Redis客户端

  1. cd C:\redis
  2. redis-cli.exe -h 127.0.0.1 -p 6379

Redis配置

1.Redis的配置文件redis.conf在Redis的安装目录下
2.查看配置，通过CONFIG GET命令实现。例如

  1. 127.0.0.1:6379> CONFIG GET port

得到redis的监听端口为6379

  1. 1) "port"
  2. 2) "6379"

3.更改配置，通过CONFIG SET命令实现。例如

  1. 127.0.0.1:6379> CONFIG SET timeout 500
  2. OK

设置了客户端闲置500s后关闭连接
4.几个典型配置选项(括号内加粗为默认项)

  • daemonize(yes/no)：redis是否以守护进程方式运行
  • port(6379)：redis的监听端口
  • timeout(300)：客户端无响应相应秒数后自动断开连接，设置为0关闭功能
  • masterauth("")：设置客户端连接服务端的密码，默认为无密码

在配置文件中有对各个配置选项的具体解释，有需要其他配置可打开redis.conf查看

Redis的使用

Redis中的命令无所谓大小写。

简单命令

Redis中数据以键值对形式存储。
1.增。
增加键值对["teacher", "LijunChen"]

  1. 127.0.0.1:6379> set teacher LijunChen
  2. OK

2.查。
查找key:"teacher"对应的value

  1. 127.0.0.1:6379> get teacher
  2. "LijunChen"

3.删。
删除key:"teacher"对应的键值对

  1. 127.0.0.1:6379> del teacher
  2. (integer) 1

哈希

Redis中的hash是键值对映射表，可用于存储对象。其相关命令以"H"开头。
1.增。
增加对象“大数据管理技术课程”

  1. 127.0.0.1:6379> hmset course name BigDataManagement teacher LijunChen time Monday 
  2. OK

2.查。

  1. 127.0.0.1:6379> hgetall course
  2. 1) "name"
  3. 2) "BigDataManagement"
  4. 3) "teacher"
  5. 4) "LijunChen"
  6. 5) "time"
  7. 6) "Monday"

列表与集合

1.列表。
相关命令以"L"开头。例如，通过lpush向列表中增加元素，lindex查找相应下标元素，llen显示列表大小。

  1. 127.0.0.1:6379> lpush student ZhangSan
  2. (integer) 1
  3. 127.0.0.1:6379> lpush student LiSi
  4. (integer) 2
  5. 127.0.0.1:6379> lpush student WangWu
  6. (integer) 3
  7. 127.0.0.1:6379> lindex student 1
  8. "LiSi"
  9. 127.0.0.1:6379> llen student
  10. (integer) 3

2.集合
集合有关命令以"S"开头，如sadd(添加元素)、smembers(返回所有元素)

  1. 127.0.0.1:6379> sadd a 123
  2. (integer) 1
  3. 127.0.0.1:6379> sadd a 456
  4. (integer) 1
  5. 127.0.0.1:6379> sadd a 987
  6. (integer) 1
  7. 127.0.0.1:6379> smembers a
  8. 1) "123"
  9. 2) "456"
  10. 3) "987"

3.有序集合
有序集合有关命令以"Z"开头，如zadd(添加元素)、zrange(返回分数区间内的元素)

  1. 127.0.0.1:6379> zadd grade 100 Zhang3
  2. (integer) 1
  3. 127.0.0.1:6379> zadd grade 98 Li4
  4. (integer) 1
  5. 127.0.0.1:6379> zadd grade 61 Wang5
  6. (integer) 1
  7. 127.0.0.1:6379> zadd grade 59 Li4
  8. (integer) 0
  9. 127.0.0.1:6379> zrange grade 0 100 withscores
  10. 1) "Li4"
  11. 2) "59"
  12. 3) "Wang5"
  13. 4) "61"
  14. 5) "Zhang3"
  15. 6) "100"

以下Java和Python中使用Redis仅为范例参考，完成上机目标不限定具体语言和工具。

Java中使用Redis

1.首先需要下载驱动包 Jedis.jar
2.在Java的Classpath中包含该驱动包的路径，或在IDE构建项目后导入该包
3.在你的Java项目中运行如下代码(注意:确保你的Redis服务端正常运行中)

  1. import redis.clients.jedis.Jedis;
  2. import java.util.List;
  4. public class TryRedis {
  5.     public static void main(String[] args) {
  6.         //连接本地Redis服务
  7.         Jedis jedis = new Jedis("localhost");
  8.         System.out.println("连接成功");
  9.         //查看服务运行状态
  10.         System.out.println("服务正在运行: "+jedis.ping());
  12.         //添加键值对
  13.         jedis.set("teacher", "LijunChen");
  14.         //查找键值对
  15.         System.out.println("key:\"teacher\"对应的值为: "+ jedis.get("teacher"));
  17.         //构建列表
  18.         jedis.lpush("student", "ZhangSan");
  19.         jedis.lpush("student", "LiSi");
  20.         jedis.lpush("student", "Wangwu");
  21.         // 获取列表数据并输出
  22.         List<String> list = jedis.lrange("student", 0 ,2);
  23.         for(int i=0;i<list.size(); i++){
  24.             System.out.println("第"+i+"个学生为:"+list.get(i));
  25.         }
  26.     }
  27. }
